# PomoFocus 🍅

A vibrant, colorful, and fun Pomodoro study timer designed to make productivity feel like a breeze.

## 📝 Original Prompt
The entire application was generated based on this request:
> "I want you to create a web app for a study pomodoro timer that allow me to change the focus time in minutes and the rest time from 5 minutes to 30 minutes while enabling me to cancel both at any time I want make sure the web app is colorful and fun"

## ✨ Features
- **Customizable Intervals**: Set your focus time anywhere from 1 to 60 minutes and rest time from 5 to 30 minutes.
- **Dynamic UI**: A glassmorphic design with animated background blobs that shift colors.
- **Full Control**: Start or cancel your sessions at any time with a single click.
- **Responsive Design**: Works beautifully on both desktop and mobile devices.
